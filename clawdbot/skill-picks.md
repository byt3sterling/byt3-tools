# Skill picks (from awesome-moltbot-skills)

Source list: https://github.com/VoltAgent/awesome-moltbot-skills

I’m not copying the whole list in here. Just the ones that actually help with **hardening**, **debugging**, and **operating** a Clawdbot/Moltbot setup.

## Security / safety

- **skills-audit** — audit locally installed skills for security/policy issues
  - https://github.com/moltbot/skills/tree/main/skills/morozred/skill-audit/SKILL.md

- **clawdbot-logs** — read/analyze Clawdbot logs + diagnostics
  - https://github.com/moltbot/skills/tree/main/skills/satriapamudji/clawdbot-logs/SKILL.md

- **linux-service-triage** — common Linux service issues (logs, systemd/PM2, permissions)
  - https://github.com/moltbot/skills/tree/main/skills/kowl64/linux-service-triage/SKILL.md

## Ops / maintenance

- **auto-updater** — auto-update Clawdbot + installed skills daily
  - https://github.com/moltbot/skills/tree/main/skills/maximeprades/auto-updater/SKILL.md

- **clawdhub** — search/install/update/publish skills via ClawdHub CLI
  - https://github.com/moltbot/skills/tree/main/skills/steipete/clawdhub/SKILL.md

- **simple-backup** — backup workspace/state locally
  - https://github.com/moltbot/skills/tree/main/skills/vacinc/simple-backup/SKILL.md

## Browser / verification / automation (use carefully)

- **verify-on-browser** — CDP browser control (good for “click this button” workflows)
  - https://github.com/moltbot/skills/tree/main/skills/myestery/verify-on-browser/SKILL.md

- **agent-browser** — headless browser automation (good for repeatable tasks)
  - https://github.com/moltbot/skills/tree/main/skills/matrixy/agent-browser-clawdbot/SKILL.md
