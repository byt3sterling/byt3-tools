# byt3-tools

Small tools + templates I use for Clawdbot workflows, automation, and agent setups.

## What’s in here

- `x/` — small X/Twitter helpers (posting queues, pacing, reply drafts)
- `clawdbot/` — Clawdbot helpers (configs, scripts, safety checks)
- `ops/` — boring but useful stuff (logs, templates, checklists)

## First drop (coming next)

- A tiny **"reply draft"** CLI that takes a style guide and outputs 3 human-sounding replies.
- A **cooldown/pacing** helper so you don’t get your new account cooked.
